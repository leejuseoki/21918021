1. PYTHON이란

파이썬은 1991년 네덜란드계 소프트웨어 엔지니어인 귀도 반 로섬이 발표한 고급 프로그래밍 언어로, 플랫폼에 독립적이며 인터프리터식, 객체지향적, 동적 타이핑 대화형 언어이다.

2.특징

많은 프로그래밍 언어들은 구문이 끝날 때 ;(세미콜론)을 붙여야 합니다. 하지만 파이썬은 세미콜론을 붙이지 않습니다.
불은 True, False로 표현합니다. 특히 비교 연산자, 논리 연산자의 판단 결과로 True, False를 사용합니다.
문자열은 ' '(작은따옴표) 또는 " "(큰따옴표)로 묶어서 표현합니다.
여러 줄로 된 문자열은 '''(작은따옴표 3개)로 시작해서 '''로 닫거나 """(큰따옴표 3개)로 시작해서 """로 닫아서 표현합니다.

4. 등산 시간 계산

풀이보기(클릭) 
                                    
~~~python
import math

a = 10
b = math.sqrt(3**2 + 4**2)
c = math.sqrt(3**2 + 4**2)
d = 8

speedNormal = int(input("평지의 속도 : "))
speedUp = int(input("오르막의 속도 : "))
speedDown = int(input("내리막의 속도 : "))

timeA = int(round(a / speedNormal,2) * 60)
timeB = int(round(b / speedUp,2) * 60)
timeC = int(round(c / speedDown,2) * 60)
timeD = int(round(d / speedNormal,2) * 60)


print("A구간은",timeA,"분")
print("B구간은",timeB,"분")
print("C구간은",timeC,"분")
print("D구간은",timeD,"분")

total = timeA + timeB + timeC + timeD

hour = total // 60
minu = total - hour* 60

print("총",hour,"시간",minu,"분이 소요됩니다.")
~~~
